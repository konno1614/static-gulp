# static-gulp

## 開発

### ディレクトリ構造

```txt
├── public  # ドキュメントルート
└── src     # コンパイル前ソース
```

#### html,css,js
`/src`内で元ファイルを編集してコンパイル。

### ローカル開発


#### nodeバーション
```
node.js(version: v16.9.1)
```

#### 依存ファイルインストール
```
npm install
```

#### 開発
```
npx gulp
```
