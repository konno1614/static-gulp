# ejs-template

## 開発

### ディレクトリ構造

```txt
├── public  # ドキュメントルート
└── src     # コンパイル前ソース
```

#### html,css,js
`/src`内で元ファイルを編集してコンパイル。

### ローカル開発

##### 依存ファイルインストール
```
npm install
```

##### 開発
```
npx gulp
http://localhost:3000/
```
